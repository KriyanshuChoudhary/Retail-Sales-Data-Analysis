# Retail Sales Analytics
An end-to-end **Retail Sales Analytics** project built using **Python, SQL Server, and Power BI**. This project demonstrates the complete data analytics workflow—from data cleaning and exploratory data analysis to SQL-based business analysis and interactive dashboard development.

---

## Project Overview
The objective of this project is to analyze retail customer shopping data and uncover meaningful business insights related to sales performance, customer behavior, product performance, and purchasing trends.

The project follows a complete analytics pipeline:

**Raw Data → Python (Data Cleaning & EDA) → SQL Server (Business Analysis) → Power BI (Interactive Dashboard)**

---

## Tech Stack

| Tool | Purpose |
|------|---------|
| Python (Pandas, NumPy) | Data Cleaning & Preprocessing |
| SQL Server | Business Analysis & KPI Calculation |
| Power BI | Dashboard Development |
| DAX | Measures & Calculated Columns |

# Project Workflow

### Data Cleaning (Python)

- Imported the retail sales dataset using Pandas.
- Checked data types and column information.
- Identified and handled missing values.
- Performed descriptive statistical analysis.
- Conducted exploratory data analysis (EDA).
- Exported the cleaned dataset for SQL analysis.

---

### Business Analysis (SQL Server)

Performed SQL analysis to answer key business questions:

- Total Revenue
- Average Order Value
- Customer Count
- Average Customer Rating
- Revenue by Category
- Top Selling Products
- Customer Segmentation
- Purchase Frequency Analysis
- Payment Method Analysis
- Shipping Type Analysis

---

### Dashboard Development (Power BI)

Built a **2-page interactive dashboard** with dynamic filters, KPI cards, DAX measures, and business-focused visualizations.

### Sales Overview

- Revenue KPIs
- Revenue by Category

- Shipping Type Analysis

### Product & Customer Insights

- Best Selling Product

---

# Key Insights

- Generated **$233,081** in total revenue from **3,900 customer transactions**.
- Achieved an **Average Order Value of $59.76**.
- **Clothing** was the highest-performing category, generating **$104,264** in revenue (~45% of total revenue).
- **Blouse** emerged as the best-selling product with **$10.4K+** in revenue.
- **Gloves** received the highest average customer rating (**3.86 / 5**).
- Customers purchasing **Every 3 Months** represented the largest customer segment (**584 customers**).
